# Tagore - Virtual Research Collaboration Ecosystem
This project proposes the design and development
of a Unified Research Collaboration Platform, an integrated digital ecosystem
that combines tools for:
* Literature Search
* Project Management
* Communication
* Funding
* and Collaborative Development across all research domains.

## Tech Stack
### Frontend Technologies
`React.js` `Axios` `Zustand`

### Backend Technologies
`Node.js` `Express.js` `MongoDB` `JWT` `Bcrypt` `Joi`
